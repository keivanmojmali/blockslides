## What's included

- **Headless core** (`@blockslides/core`): editor runtime built on ProseMirror
- **Framework bindings**: React (`@blockslides/react`) and Vue 3 (`@blockslides/vue-3`)
- **50+ extensions**: text, media, layouts, tables, math, markdown, and more
- **Import/export helpers**: HTML, Markdown, and static rendering
- **AI-ready schemas**: templates and structured data for AI workflows
